# Employee Shift Scheduling System

## Overview

This project is a Python-based employee scheduling application. The company operates seven days a week and offers three shifts each day:

- Morning
- Afternoon
- Evening

The program stores employee preferences, assigns shifts, resolves conflicts, and generates a weekly schedule.

## Features

- Collects employee names and shift preferences
- Uses loops, conditionals, and branching statements
- Limits employees to a maximum of 5 work days per week
- Prevents employees from working more than one shift per day
- Assigns employees to available shifts
- Generates a readable weekly schedule

## Requirements

Python 3.x

## How to Run

Open a terminal in the project folder and run:

```bash
python3 scheduler.py
```
