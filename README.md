# Employee Shift Scheduling System

## Overview

This project implements an employee shift scheduling system using two programming languages: Python and Java. The company operates seven days a week and offers three shifts per day: Morning, Afternoon, and Evening.

The application collects employee shift preferences, assigns employees to available shifts, resolves scheduling conflicts, and generates a final weekly schedule. The project demonstrates the use of control structures such as loops, conditional statements, branching, and data structures in multiple programming languages.

## Features

* Stores employee names and preferred shifts
* Creates a weekly schedule for seven days
* Uses loops, conditionals, and branching statements
* Prevents employees from working more than one shift per day
* Limits employees to a maximum of five working days per week
* Assigns employees to available shifts based on preferences
* Resolves scheduling conflicts when preferred shifts are unavailable
* Generates a readable weekly schedule
* Implemented in both Python and Java

## Project Files

* `scheduler.py` – Python implementation of the scheduling system
* `ShiftScheduler.java` – Java implementation of the scheduling system
* `README.md` – Project documentation and execution instructions

## Requirements

* Python 3.x
* Java Development Kit (JDK 8 or higher)

## Running the Python Version

Open a terminal in the project directory and run:

```bash
python3 scheduler.py
```

## Running the Java Version

Compile the Java source code:

```bash
javac ShiftScheduler.java
```

Run the program:

```bash
java ShiftScheduler
```

## Output

The program displays the final employee schedule for each day of the week and lists the employees assigned to the Morning, Afternoon, and Evening shifts.
